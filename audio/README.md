# Música de Fundo

Coloque seu arquivo de música nesta pasta com o nome: **background-music.mp3**

## Formatos Suportados

- MP3 (recomendado)
- WAV
- OGG
- M4A

## Como Adicionar Música

### Opção 1: Arquivo Local
1. Coloque o arquivo MP3 nesta pasta
2. Renomeie para: `background-music.mp3`
3. Pronto! O site carregará automaticamente

### Opção 2: URL Online
1. Abra o arquivo `index.html`
2. Procure pela tag `<audio>`
3. Altere o `src` para a URL de sua música:

```html
<audio id="bgMusic" loop preload="none">
    <source src="https://exemplo.com/sua-musica.mp3" type="audio/mpeg">
</audio>
```

## Recomendações de Músicas

Para um clima emocional e cinematográfico, procure por:

### No YouTube
- "Cinematic Piano"
- "Ambient Electronic Music"
- "Lo-fi Hip Hop"
- "Classical Ambient"

### Plataformas Royalty-Free
- Epidemic Sound
- Artlist
- AudioJungle
- Incompetech
- YouTube Audio Library

## Dicas

- Escolha uma música **suave** e **instrumental**
- Duração ideal: **3-5 minutos**
- Volume: Não muito alto (o site toca a 50% do volume)
- Formato: MP3 recomendado para melhor compatibilidade

---

**Nota**: A música só toca após o usuário clicar no botão "Clique" (por política de navegadores modernos).
